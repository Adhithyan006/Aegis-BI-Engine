#  Aegis BI Engine: Agentic Intelligence System

![Python](https://img.shields.io/badge/Python-3.9+-blue.svg)
![Streamlit](https://img.shields.io/badge/Streamlit-1.25+-red.svg)
![CrewAI](https://img.shields.io/badge/AI-CrewAI-gold.svg)

A high-fidelity Multi-Agent Business Intelligence system designed to orchestrate autonomous researchers and data architects for specialized domain analysis.

##  Key Features
| Feature | Description |
| :--- | :--- |
| **Agentic Synthesis** | Autonomous agents research and generate high-fidelity datasets. |
| **Geo-Validation** | Built-in geography guard to ensure 100% data accuracy (e.g., TN Districts). |
| **Royal UI** | Premium Gold & Sandalwood themed dashboard for executive reporting. |
| **BI Integration** | One-click export of Power BI-ready structured CSVs. |

##  Tech Stack
- **Orchestration:** CrewAI (Multi-Agent Systems)
- **Engine:** Groq (Llama-3.3-70B)
- **Frontend:** Streamlit (Custom CSS Architecture)
- **Data:** Python Pandas & Logic Guards
